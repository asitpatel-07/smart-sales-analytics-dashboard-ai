# Smart Sales Analytics Dashboard AI

An AI-powered sales analytics dashboard built using **Python**, **Streamlit**, and **Machine Learning** to track sales performance, analyze profit/loss, and predict future sales trends.

## 🚀 Features

- Add and manage sales records
- Track product-wise sales performance
- Profit / Loss analysis
- Interactive dashboard view
- Sales data table with structured output
- AI-based future sales prediction
- Clean and user-friendly interface

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- Scikit-learn
- NumPy
- CSV Data Handling

## 📂 Project Structure

- `app.py` → Main Streamlit application
- `data.csv` → Sales dataset
- `requirements.txt` → Project dependencies
