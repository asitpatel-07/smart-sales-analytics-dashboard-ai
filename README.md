# 📊 Smart Sales Analytics Dashboard with AI Prediction

An interactive **Data Analytics + AI Dashboard** built using **Python, Streamlit, Pandas, Plotly, and Machine Learning**.  
This project allows users to enter real-time sales data, track profit/loss, visualize business performance, and predict future sales using AI.

---

## 🚀 Live Demo
🔗 [Open Live App](https://sales-ai-dashboard.streamlit.app)

---

## 📌 Features

- 📥 **Manual Sales Data Entry**
  - Add product sales data by date
  - Enter sales amount and profit/loss values

- 📋 **Sales Data Table**
  - Displays all entered records
  - Row numbering starts from 1 for better readability

- 🗑 **Delete Wrong Entry**
  - Remove incorrect data entries easily

- 📅 **Date Filter**
  - View analytics for:
    - All Dates
    - Selected Date

- 💰 **Profit / Loss Tracking**
  - Separate calculation for:
    - Total Profit
    - Total Loss
    - Net Result

- 📈 **Interactive Visualizations**
  - Sales Trend Chart
  - Sales by Product Chart
  - Profit vs Loss Chart

- 🧠 **Business Insights**
  - Top Selling Product
  - Highest Profit Product
  - Loss Detected Products
  - Business Status (Profit / Loss)

- 🤖 **AI Sales Prediction**
  - Predict future sales using **Linear Regression**
  - Choose number of days to forecast

- 📥 **CSV Download**
  - Download filtered sales data
